# Fireworks Exhibitor Exam Prep
Ohio Type I Fireworks License Exam preparation tool
100 questions covering NFPA 1123, NFPA 1126, Ohio Law (ORC 3743), safety procedures, and terminology.

## Features
- 70 flashcards covering all exam topics
- Flashcard mode (tap to reveal)
- Quiz mode (multiple choice with instant feedback)
- Category filters
- Streak tracking
- Pass predictor (70% required)
- Review missed questions

## How to Use
1. Open index.html in a browser
2. Start with flashcard mode to learn content
3. Take a full quiz to test your knowledge
4. Review any missed questions

## Exam Details
- 100 questions
- 120 minutes
- 70% passing score (70 correct)
- Administered by PSI Services
